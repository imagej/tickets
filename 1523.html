Allow any RegionOfInterest to be wrapped by a PointSet

Reported by: bdezonia Owned by: bdezonia

Severity: serious Keywords:

Blocking: #1511

Description (last modified by bdezonia)

It would be useful to ImageJ if a RegionOfInterest could be wrapped by a PointSet. Allow this in Imglib2 OPS.

By wrapping we would have a discrete view of a continuous region. One would not rely on measurements made on such a PointSet. But one could sample and iterate over an arbitrary region using a fixed grid approach.
