Description

From David Knecht:

There is an interface inconsistency between LOCI/ImageJ Open and the Import Image Sequence commands. The latter works in a way that is confusingly different to me. It requires highlighting the folder rather than the file and has led me to frequently believe that I could not open a set of images because they were grayed out in the Finder. I think the interface is cleaner if everything works in internally consistent ways.

I agree with Dave's assessment but there is little we can do about the File > Import > Image Sequence command of ImageJ 1.x. We are striving to make ImageJ's built-in File > Open command use SCIFIO out of the box, and eventually provide a richer set of options when opening images, which should ultimately obviate the need for File > Import > Image Sequence altogether.
