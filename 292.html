Description

 Omero Tables is a mechanism for working with column-based data within OMERO. Take a look and determine whether this functionality could be useful for CPA/OMERO integration.

This is in part of our effort to research potential integration of OMERO with CellProfiler Analyst and other (Python-based) CP code.

from  http://trac.openmicroscopy.org.uk/ome/wiki/OmeroTables#Going-forward:
"The Tables API itself provides little more than a remotely accessible store, think of it as a server for Excel-like spreadsheets"

After speaking with Caterina Strambio De Castillia, it sounds like users could benefit if we were to attach CSVs with analysis results generated by CP to each image in OMERO. Emailing omero-devel for more info on how easy this is and whether it's safe from potential changes in the API.
