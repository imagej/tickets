UI ideas for Overlay Manager

Cc: Blocked By:

Description (last modified by bdezonia)

THe IJ2 Overlay Manager is closely modeled after the IJ1 Roi Manager. It would be nice if we improved the UI a bit. One improvement would be to have three panes. The left pane could show the name/XY location of all the overlays. The middle pane could contain the attributes of the overlay selected in the left pane and ideally they could be edited in place. The right pane could contain the push buttons and popup menus that implement the Overlay Manager commands like Delete, Draw, etc.
