Provide a UI to customize menu items

Severity: minor Keywords:
Cc: bdezonia, curtis Blocked By:

Description

While fixing ticket #1795, the issue was discussed what to do about hiding certain menu items if the user does not want to see them.

As this issue is not limited to LUTs, and not only to hiding but also touches moving things around between menus and also renaming menu item labels, it needs to be solved more generically, in the menu service.

A lot of care has to be taken here in light of retaining the ability to record macros and scripts: the renaming/hiding must not break the event recorder.

It would be nice to be able to customize, but I see it as similar to internationalization, in that it is more about convenience than functionality.
