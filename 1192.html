Some bugs crop up occasionally related to this. For instance #1605 would have been entirely avoided by locking. The fact that we harmonize all displays with their imagepluses every time we run a legacy plugin means that if we run multiple legacy plugins simultaneously we are potentially munging the same data simultaneously too. This is a problem that needs addressing.
