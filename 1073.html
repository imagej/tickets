Drawing tools don't always work on Linux

Reported by: bdezonia Owned by: bdezonia
Priority: major Milestone: imagej2-b6-legacy
Component: Core Version:
Severity: serious Keywords:
Cc: Blocked By:
Blocking: #1571

Description

The drawing tools (brush and pencil) are designed to draw using the fg channel values unless the ALT key is down and then it draws using the bg channel values.

On Ubuntu 11 with Unity this can't work. When you click mouse with ALT down Unity thinks you are initiating a drag window event. So on stock Ubuntu 11 one cannot draw using background values.
