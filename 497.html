Rotate 90 degrees: choose pan/zoom behavior

Reported by: bdezonia Owned by: curtis
Priority: minor Milestone: imagej-2.0.0
Component: Core Version:
Severity: serious Keywords:
Cc: Blocked By:
Blocking: #1439

Description

If panned and zoomed on an image and then you invoke a rotate 90 degrees it resets pan to 0,0 and zoom to 1.0. Maybe it should resize the panel as needed but set zoom so that old view center is once again the view center (those coords need to be transformed of course) and that the zoom level is the same as before.
