Description

Recently display refactoring was done shortly before the release of beta2. After those changes images always open at 100% scale. This is a problem for large images. Also some images (like Embryos and some of the About images) don't fit on a typical laptop screen.

Note that code inspection shows that setInitialScale() is no longer called by anyone.
