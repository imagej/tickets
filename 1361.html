Description

Since the switch from the Fiji Updater to the ImageJ Updater, Fiji no longer ships any non-trivial uploader by default.

To help the situation, let's special-case the ssh uploader and try to install it and use it in a custom class loader so that people are not surprised by not being able to upload at all any longer.
