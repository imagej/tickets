Linked histogram tools for derived images

Severity: serious Keywords:
Cc: Blocked By:

Description

SLIM Plugin has the concept of a set of derived or fitted images, such as tau or chi-square, derived from an initial lifetime image. The histogram tool in SP supports dragging or entering a range of min/max LUT range values. It also supports hiding pixels outside of this LUT range on the tau image and having them also disappear from the chi-square image. Similarly the user can also hide pixels outside of LUT range on the chi-square image and have them disappear from the tau image. This allows the user to "drill down" into the data during post-fit analysis by discarding outliers from each fitted image. The user could say I am only interested in pixels within chi-square range 0.75..1.25 and tau range 0.5..0.75, for example.
