Overlay Manager AND/OR/XOR display issues

Reported by: bdezonia Owned by: bdezonia
Priority: major Milestone: imagej2-b9-rois
Component: UI Framework Version:
Severity: serious Keywords:
Cc: Blocked By:
Blocking: #285

Description

When you do a boolean operation in the Overlay Manager the selection border stays a rectangle. And if you don't have the fill overlay property set to something other than invisible you cannot see that the logical operations really are working. We need to improve the rendering of composite overlays so their selected region outlines the displayed region rather than the union of the original rectangular areas.
