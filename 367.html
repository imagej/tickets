We are the process of migrating all of LOCI's open source code to GitHub. As of this writing,  19 projects have been migrated. This change moves us in the opposite direction: splitting repositories rather than combining them, but with the Git/GitHub workflow, it makes sense, especially with a Maven-based build system.

As such, I am repurposing this ticket to describe that work, particularly what still remains to be done.

The following projects still need to migrate:

  •  curve-fitter
  •  flow-cytometry
  •  loci-checks
  •  misc-plugins
  •  slim-plotter
  •  slim-plugin
  •  visbio
  •  visbio-plugins
  •  jvmlink
  •  multi-lut
  •  JEX

After that, we can repurpose the LOCI Software Trac as a single multi-project Trac.
