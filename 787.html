Description

This ticket covers the following commands
Image > Type > 8-Bit
Image > Type > 16-Bit
Image > Type > 32-Bit
Image > Type > 8-Bit Color
Image > Type > RGB Color
Image > Type > RGB Stack
Image > Type > HSB Stack

When running any of the above commands the display window is duplicated.

Note that ImageJ Menus (in the MenuBar) are visible when the new display is active but not when the original display is active.

Sometimes one or two extra displays titled "outputs" are created with channel sliders and no image. When the channel sliders are moved, they typically affect the original image display.

tested on b0a67146225eaa9a3db23ce2b4564b2971bad4d6
