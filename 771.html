Description

Try opening the mitosis 5d sample and running reduce dimensionality on it. You'll find that it mostly works except it does not respect the currently selected plane when fixing dimensions.

eg: if you scrub to Channel=2, Z=5 and run reduce dimensionality flattening these dimensions, the resultant image will contain all timepoints from Channel=1, Z=1.
