Generalize the auto-installing of ij-updater-ssh

Reported by: dscho Owned by: dscho
Priority: major Milestone: imagej2-b7-ndim-data
Component: Updater Version:
Severity: minor Keywords:
Cc: Blocked By:
Blocking: #1737, #1901

Description

Now that we have a WebDAV backend, people will want to use it, but be confused when it is not available because ij-updater-webdav is not installed.

Generalize the code that detects when "ssh" is not available (extending it to special-case "sftp" to mean the same as "ssh") to try to install ij-updater-<protocol> and use it right away.

While at it, make it work with initializing new update sites, too.
