Description

This tool gives the ability to set the LUT range automatically to min/max, zoom into quartile range, or manually drag or type range min/max. For a set of fitted images it gives the ability to hide pixels out of LUT range (set all fitted values to NaNs). It also adds smoothing capabilities.
