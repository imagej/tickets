Description

The current idea is to bring the selection to the front upon activation; Ctrl+click (which should be mentioned in the status bar when hovering over a hidden overlay) could activate "lower" overlays.

The complete control should be available via an overlay manager, i.e. a list where the user can drag 'n drop overlays to a new position, as well as move the selected list items via arrow buttons.
