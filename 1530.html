Description

The legacy Monitor Events plugin works but does not have the same output as IJ1. This is because the legacy layer does certain things with IJ1. We likely need a pure IJ2 plugin to watch IJ2 events.

Example:
open boats
start Monitor Events command
open another sample image
close that image
close boats

do this in IJ1 and IJ2 and see that the event histories are a little different
