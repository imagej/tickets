Reported by: bdezonia Owned by: bdezonia

Component: Plugins Version:
Severity: serious Keywords:
Cc: Blocked By:

Description

The legacy service has been enhanced with a runLegacyPlugin() command. Use this command as needed to rearrange where some IJ1 plugins reside in the menus. We could define pure IJ2 plugins that simply call runLegacyPlugin() but locate themselves in other places in the menu. We'd need to provide code to surgically remove the old legacy plugin location from the menu too.
