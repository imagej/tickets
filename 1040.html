Description

Just another bit of infrastructure that should move from Fiji to ImageJ2: the sample file uploader. It could be enhanced just like the Updater to allow for several backends, and it could also add the ability to specify not only between pre-selected sites but also free-form URLs, probably after adding support for FTP via jFTP.

Of course, Fiji would still retain its Help>Upload Sample Image, but use the ImageJ2 base plugin to do the heavy-lifting.
