Description

IJ1 has some support for HSB images. You can take a RGB image, use Image::Type::HSB Stack menu entry to make a 3 channel stack of Hue, Sat, and Bri. The choose Image::Type::ColorRGB and it turns back to color. There is special case code in IJ1 that detects if a 3 channel stack has slice labels "Hue" etc. and does appropriate things.

Determine how we want to support HSB color model going forward.
