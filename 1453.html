Fully document ImageJ 1.x compatibility mode

Reported by: bdezonia Owned by: bdezonia
Priority: major Milestone: imagej2-b5-scripting
Component: Plugins Version:
Severity: serious Keywords:
Cc: Blocked By:
Blocking: #1508

Description (last modified by bdezonia) (diff)

Recently it was decided to support ImageJ 1.x compatibility mode in the OptionsCompatiblity options plugin class. In order to not confuse the end users we should enhance the dialog somewhat.

First a short textual description should be added to the dialog. It should also include a link to a web page that discusses all the ways that IJ2 differs from IJ1 when modern mode is enabled.

Note that there is a wiki page devoted to this dialog already at  http://wiki.imagej.net/ImageJ2/Documentation/Edit/Options/Compatibility
