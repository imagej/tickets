Description

The legacy version of select none does not work correctly. This command is usually blacklisted but would be accessible via legacy macros.

Here is what is broken:

1) If multiple overlays are selected in the view then the active overlay is removed rather than all removed.

2) If there is one overlay that fills view (via Select All for instance) the overlay is not removed.
